# LWJGL-Voxel-Project 🧊

### Moteur de rendu 3D haute performance en Java
Ce projet consiste en la création d'un moteur de type "Voxel Engine" (type Minecraft) pour approfondir mes connaissances en gestion de mémoire et rendu graphique complexe.

---

### 🚀 Caractéristiques Techniques
**Rendu Graphique :** Utilisation d'**OpenGL** via la bibliothèque **LWJGL**.
**Optimisation :** Implémentation d'un système de **Chunks** pour la gestion efficace de l'affichage et de la géneration.
**Langage :** Entièrement développé en **Java**.

### 🛠️ Installation et Utilisation
1. **Clonage :** `git clone https://github.com/pepin2pomme/LWJGL-Voxel-Project.git`
2. **Dépendances :** Les fichiers `.jar` nécessaires sont situés dans le dossier `/libs`.
3. **Exécution :** Lancer la classe principale depuis votre IDE (IntelliJ recommandé)
